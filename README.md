# Nirvalla Tasks
A personal-first, open-source task management app.

**Nirvalla Tasks** is a modern, minimal, and powerful task management app inspired by apps like Google Tasks, Todoist and Trello. It has an intuitive UI, and a focus on **simplicity, speed, and personal data control**.

### Features:

- **Task Management** – Create, update, complete, and reorder tasks.
- **Task Lists** – Organize tasks into tasklists (Kanban Board like).
- **Fast & Minimal** – Built with React, Next.js, and a smooth drag-and-drop experience.
- **Drag & Drop** – Uses **DnD-Kit** for intuitive reordering.
- **Sync API** – REST API with secure JWT-based authentication, Syncs your tasks so you can access them from anywhere.
-  **Google & GitHub OAuth** - For easier login.
-  **Sound Effects** – Subtle task completion sounds for a satisfying UX.

### Tech-Stack
**Frontend:**

- Next.js (React)
- Tailwind CSS
- ShadCN/UI
- DnD-Kit
- Axios

**Backend:**

- Node.js + Express
- MongoDB
- Passport.js (Google  & GitHub OAuth)
- JWT
